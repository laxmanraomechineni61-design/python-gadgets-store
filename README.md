# Python Gadgets Store

A simple console-based shopping cart application built using Python.

## Features
- Display available products
- Add items to cart
- Calculate total bill
- Simple dictionary-based inventory

## Technologies
- Python
- Dictionaries
- Lists
- Loops and Conditions

## How to Run
1. Download the file `gadgets_store.py`
2. Run using: `python gadgets_store.py`
